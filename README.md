# TUYYO CLUB — SUP Rental Costa Blanca

Веб-сайт для оренди SUP-дощок на узбережжі Коста-Бланка (Іспанія).

## Структура проєкту

```
├── index.html    # Основна розмітка HTML
├── style.css     # Стилі CSS
└── script.js     # Логіка JavaScript
```

## Розгортання

Відкрийте `index.html` у браузері або розгорніть на будь-якому статичному хостингу (GitHub Pages, Netlify, Vercel).

## Функціональність

- Багетапна форма бронювання
- Канвас для електронного підпису
- Адаптивний дизайн (mobile-first)
- Акордеон FAQ
- Мобільна навігація
- Плавна прокрутка
